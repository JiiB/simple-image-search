# vue-image-search

## Project setup
Create a `.env` file and add your **Pixabay** API key

```
VUE_APP_API_KEY=<your-api-key>
```

Install the node_modules

```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
